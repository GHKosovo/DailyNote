# JDK类库的根类：Object

任何一个类默认继承Object，就算没有直接继承，最终也会间接继承

## Object的几个常见的方法

| 方法       | 描述                     |
| ---------- | ------------------------ |
| clone()    | 负责对象的克隆           |
| hashCode() | 获取对象哈希值的一个方法 |
| tostring   | 将对象转换为字符串形式   |
| finalize() | 垃圾回收器负责调用的方法 |
| equal()    | 判断两个对象是否相等     |
