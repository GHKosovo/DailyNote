## 概述

String 表示字符串类型，在java中随便使用双引号括起来的都是String对象，Java中规定双括号括起来的字符串是不可变的，也就是说这个字符串一直都存在，不能更改。这些字符串都是直接存储在方法区的"**字符串常量池**"中。

为什么sun公司把字符串存储在“**字符串常量池**”呢？因为i字符串在实际开发中使用太频繁。为了执行效率，所以这么干。

## 示例

```java
//main方法内
//直接复制
String str0 = "abc"+"ef"
String str1 = "abc"
//新建变量
String str2 = new String("abc")
```
